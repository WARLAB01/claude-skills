## Template Usage Notes

### Required Fields

All character sheets MUST include:
- Full name and aliases
- Role classification
- First appearance
- Physical appearance (at least overview)
- Core personality traits with evidence
- Relationships (at least one category)
- Scene appearances list

### Optional Fields

Include if applicable:
- Voice & Communication (required if character speaks)
- Arc & Development (required for main characters)
- Role in Mystery (only if mystery mode enabled)
- Visual Prompt (recommended for main characters)

### Evidence Requirements

Every claim about the character must include:
- Source Unit_ID
- For significant claims: direct anchor quote

### Updating During Extraction

As new chunks are processed:
1. Add new details to appropriate sections
2. Note source Unit_ID for each addition
3. Check for conflicts with existing entries
4. Update scene appearances list
5. Revise arc section if character development occurs
