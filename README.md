## Installation

Copy the desired skill folder(s) into your Claude skills directory:

```
~/.claude/skills/          # Claude Code
~/.skills/skills/          # Claude Cowork
```

Each skill contains a `SKILL.md` file with metadata and instructions, plus optional `references/` directories with supporting documentation.
